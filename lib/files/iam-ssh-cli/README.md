# IamSsh

Create local linux users from IAM users.

## Usage

    iam-ssh iam list
    iam-ssh iam ssh_key USER
    iam-ssh user list
    iam-ssh user create

## Installation

Add this line to your application's Gemfile:

    gem "iam-ssh"

And then execute:

    bundle

Or install it yourself as:

    gem install iam-ssh
